About the project

Homework for Part 1:

  1. Install @angular/cli
  2. Generate Application by cli
  3. Generate Component
  4. Generate Service
  5. Run Build
  6. Run Start
  7. Run tests

Homework for Part 2:

  1. Create components structure accordingly to mockup with minimal styles (width, height)
  2. Use only stub data (without service)
  3. Use Input and Output for providing details like clicked item or selected object and etc
  4. Create at least two routes (for ex – home and details)

Homework for Part 3:

  1. Lifecycle Hooks – add usage of ngDestroy, ngOnInit. At least for one component log changes in console by ngOnChanges
  2. Pipes –
    • Use at least two standard pipes (https://angular.io/guide/pipes)
    • Create custom pipe (for example filter)
  3. Services – add at least one service in application
  4. HTTP – use any public API call (whether, news and etc)
  5. Use angular forms/reactive forms for adding new items from UI

Homework for Part 4:

  1. Add at least one component on UI dynamically. For example:
    • Editing panel for edit and read mode
    • Login/registration panel
    • Show/Clear any panel by button
  2. Move components on ChangeDecetionStategy.OnPush and MarkForCheck.
  3. Use at least for one component Detach/Reattach from ChangeDetectorRef

Homework for Part 5:

  1. Integrate Angular CDK
  2. Use at least 5 components from https://material.angular.io/cdk/categories (table required*)
  3. Integrate Angular Virtual Scroll.

Homework for Part 6:

  1. Integrate Angular Material
  2. Use at least 5 components from https://material.angular.io/components/categories
  3. Integrate table with Pagination, Sorting, Filter
